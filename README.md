# Ambulance traffic priority (V2I demo)

Hardcoded React + Vite demo: a stylized city map, driver HUD, dispatch strip, and a live V2I log while an ambulance runs a fixed route and traffic lights grant TSP-style preemption.

## Run locally

```bash
cd traffic-light-demo
npm install
npm run dev
```

Open the printed `http://localhost:…` URL, click **Start**, then use **Pause** / **Reset** and **0.5x–4x** speed as needed.

## Build

```bash
npm run build
```

Output is in `dist/`; serve with any static host.
